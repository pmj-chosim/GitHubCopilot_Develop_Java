# GitHubCopilot_Develop_Java  

## 1. IntelliJ에 GitHub Copilot 설치하기  

IntelliJ에서 `File` > `Settings` > `Plugins` > 검색창에 `GitHub Copilot` 검색 > `Install`(설치 버튼)을 눌러 설치   
, IntelliJ 맨 우측에 있는 GitHub Copilot 버튼을 눌러 채팅을 연 후 로그인 버튼을 눌러 로그인
